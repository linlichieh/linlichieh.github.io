Large Language Model (LLM)

Features

  • is a general term
  • is a model trained on vast amount of text data to understand and generate human-like language
  • is designed to perform tasks like text generation, question answering, translation, summarisation, etc.
  • GPT-4 and ChatGPT both are LLM

ChatGPT

Features

  • is a specific instance of an LLM based on the GPT architecture
  • is designed to generate human-like text, respond to user queries, and engage in conversation, etc.
  • can be used for building applications for natural language understanding and generation

Langchain

Features

  • not a LLM
  • an open-source library and framework
  • is designed to help developer build applications pwered by LLM
  • provides tools for
    • chaining prompts (multiple steps)
    • integrating external data sources
    • creating agents with memory and reasoning capabilities
      • reason: break down complex problems into smaller sub-tasks
      • memory: persist state or context between different interactions or calls of a chain/agent
        • this enables the agent to remember previous conversations, inputs, or actions, and use that information to provide more accurate and context-specific responses in subsequent interactions
    • enabling more advanced and context-aware applications

What are some scenarios I need to use LangChain as ChatGPT can’t achieve?

For example: creating a travel planning assistant that can

  • interact with users (based on users’ preferences and history)
  • retrieve real-time flight and hotel infomation (from external APIs)

LangChain can do

  • Create chains of LLM calls to handle multi-step tasks like understanding user queries, fetching data from external APIs, processing the data, and generating personalized recommendations.
  • Implement memory to persist user-related information across different interactions, allowing the assistant to remember previous conversations and provide relevant recommendations.

The role of ChatGPT to play here is to handle natural language understanding and generation tasks for LangChain

How to use LangChain?

  • Identify the sub-tasks that need to be performed to accomplish the main goal
  • Break down the user input or problem into smaller, manageable parts
  • Create a sequence of prompts or interactions that involve LLMs
