React.js

Develop react.js without node.js

1 ) Initialize a react app

Run a node container to install react app. It’s important to mount the current work space to container volume so that the created react app folder will be in local folder as well.

Run docker container run -it -v "$(pwd):/app" -w /app node:22 bash

-w /app = WORKDIR /app -v "$(pwd):/app" = volumes .:/app

Then you will enter the container.

Next, use the vite to create react app

Run npm create vite@latest and finish below

  1. Project name: react-app-demo
  2. Select a framework: React
  3. Select a variant: TypeScript

Run:

cd react-app-demo
npm install
npm run dev

  VITE v5.4.7  ready in 124 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help

The default port of the app is :5173

Once you see ➜ Local: http://localhost:5173/ like above in the console, it means that the react is running successfully and you can quit the container.

2 ) Create docker files in react app folder

Dockerfile:

FROM node:22
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 5173
CMD ["npm", "run", "dev", "--", "--host"]

docker-compose.yml:

services:
  react-app:
    build: .
    ports:
      - "5173:5173"
    volumes:
      - .:/app

Run docker compose up

Open your browser and open http://localhost:5173/.

If you see Vite + React in the page, it meens that you are running react on your local successfully.
